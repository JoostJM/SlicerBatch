================================================
Welcome to the SlicerCaseIterator documentation!
================================================

SlicerCaseIterator is a scripted module extension for 3D slicer.
It's purpose is to streamline the segmentation of image datasets by handling
loading and saving.

It's features include:

- Handling of volume loading
- Streamlined and standardised naming of created segmentations
- Easy reviewing of segmentations without inadvertently overwriting the originals
- Auto-updating the layout to best reflect the loaded volumes
